## 正文

剧本，是影视项目的源头，也往往是最大风险的藏身之处。长期以来，影视行业对剧本的评估高度依赖"老法师"的个人经验与漫长的人工围读。不仅投资标准难以统一，无数好故事也常常在爆满的邮箱中明珠暗投。

如今，大模型技术正在重塑这一影视工业的"第一道坎"。依托百度文心大模型ERNIE-5.1，浙江燃点创新影视科技正式推出"燃点AI剧本评估平台"。十余分钟，即可生成客观、可追溯的五维结构化剧本评估报告。它是制片人的得力"伴侣"，也是编剧创作者贴心的"好友"——陪伴你从立项之初就看清项目全貌，让好剧本更早被发现，让创作与决策都更有底气。

### 制片与编剧都需要一位"懂行的好友"

"一剧之本"，决定了影视项目的起评线。然而，在传统的影视流水线中，最核心的剧本评估环节，却正在让行业的两端——"制片投资方"与"编剧创作者"共同陷入困境：

- **制片端的困扰：项目评估缺少"靠谱参谋"**

影视圈最大的遗憾往往是"剧本阶段省小钱，制作阶段亏大钱"。长久以来，剧本评估极度依赖个人的审美与经验，将动辄千万的投资建立在缺乏系统参考的"感觉"之上。很多结构性硬伤往往要到开机后、甚至剪辑台上才暴露，造成不可挽回的巨大损失。制片人急需一位懂行、客观、随时在场的"评估搭档"。

- **编剧端的孤独：好故事缺一位"走心的读者"**

在影视公司策划和责编的邮箱里，永远躺着看不完的剧本。真正的好本子，可能仅仅因为审稿人那天状态不佳、阅读疲劳，或者只扫了前三页，就错过被看见的机会。新人编剧的心血，往往连被完整阅读和认真反馈的机会都没有。他们需要的，是一位能从头到尾细读剧本、给出多维专业反馈的"知心好友"。

行业亟需一位"不知疲倦、标准统一、没有偏见"的剧本伙伴——既懂制片人的项目焦虑，也懂编剧的创作孤独。

大模型的爆发，让这样一位"制片伴侣"与"编剧好友"成为现实。

### 燃点AI平台：你的"制片伴侣"，三步呈上专业评估

针对上述行业需求，燃点AI剧本评估平台应运而生。它像一位专业的"制片伴侣"，将极其复杂的影视专业评估逻辑封装成了极简的用户体验。用户只需三步：**上传剧本 → 填写基本信息 → 倒一杯咖啡的时间（十余分钟）**，即可收获一份详尽的评估报告。

这份报告的底层基石，紧紧围绕影视专业领域的**五大核心维度**展开：**故事架构、人物引力、台词质量、情感共鸣、市场潜力**。平台针对这五大维度进行深度"阅卷"，每个子项不仅给出精确打分与分析，更会引用剧本原文作为"打分依据"，避免"空对空"的泛泛而谈。

在此硬核的专业评估基础之上，平台将这些深奥、复杂的文本分析，转化为了**直观、可视、可感知**的决策图表：

1. **全局感知（多维雷达图）**：一键生成项目综合表现的雷达图，五大维度的长板与短板一目了然。

2. **人物关系可视化（拓扑图谱）**：将"人物引力"与"情感共鸣"的底层分析，转化为主要角色的健康度诊断与错综复杂的人物关系网络图。

3. **情绪张力可视化（节奏曲线）**：将"故事架构"中无形的起承转合，转化为可视化的"节奏曲线"，它实际上是在脑海里"用数据把电影先试映了一遍"，高潮点、平缓段落与冗余场景清晰可见。

4. **决策参考可视化（市场对标）**：基于"市场潜力"维度，智能匹配同类竞品的真实市场数据并生成对比看板。

十余分钟完成十万字的深度拆解，并生成极具专业度的五维可视化报告——支撑如此高效、精准体验的背后，是平台全流程接入的主推理引擎：**百度文心大模型ERNIE-5.1**。

- **长文本"深度共情"，拒绝空泛的概念包装**

影视剧本评估最忌讳乱用"社会议题"、"爆款逻辑"等大词进行概念化包装。ERNIE-5.1不仅具备强大的长文本理解能力，其对剧本主线和类型定位的抓取更是与剧本"内部证据"深度绑定。以院线电影《今年夏天去看海》为例，文心大模型不仅理顺了剧情，更可以领悟"去看海"背后深藏的情感主题，展现出令人会心一笑的"共情力"。

在动辄数十页的超长评估报告中，文心5.1严丝合缝地维持着"分析-依据-建议"的结构闭环，绝不出现"前面说契合、后面说不契合"的逻辑漂移。更重要的是，它的建议极具执行力：不会泛泛而谈"加强冲突、优化节奏"，而是精准引用具体场次号和人物台词，给出"删减某条故事线、压缩哪几场戏、哪个人物应提前出场"的"手术刀式"微观修改方案。

## 核心方法论提取（供项目参考）

### 评估流程

1. 上传文本（剧本/小说）
2. 填写基本信息（类型、目标受众等）
3. 大模型深度阅读分析（十余分钟）
4. 生成五维结构化评估报告

### 五维评估模型

1. **故事架构** → 情节完整性、叙事结构、节奏把控
2. **人物引力** → 角色塑造、人物弧光、关系网络
3. **台词/文笔质量** → 语言表现力、对话自然度
4. **情感共鸣** → 情感张力、读者代入、共鸣深度
5. **市场潜力** → 类型定位、竞品对标、受众分析

### 可视化组件

1. **多维雷达图**：五维综合评分
2. **人物关系拓扑图谱**：角色关系网络
3. **情绪/节奏曲线**：情节张力波形
4. **市场对标看板**：同类作品对比

### 评估报告要求

- 每个评分必须有原文引用作为依据
- 结构闭环：分析 → 依据 → 建议
- 微观修改建议需引用具体章节/段落
- 避免"空对空"的泛泛评价
