# Novel Eval 📚

AI 驱动的小说评估系统。基于大语言模型，对小说进行五维结构化评估，生成可视化评估报告。

## 灵感来源

参考 [燃点AI剧本评估平台](https://www.burning-in.com/) 的思路，将影视剧本的五维评估方法论迁移到小说领域。

## 核心设计

### 五维评估模型

| 维度 | 说明 |
|------|------|
| **故事架构** | 情节结构完整性、起承转合、叙事节奏、悬念设置 |
| **人物塑造** | 角色立体感、人物弧光、角色关系网络、动机合理性 |
| **文笔质量** | 语言表现力、对话自然度、修辞运用、风格统一性 |
| **情感共鸣** | 情感张力、读者代入感、情绪节奏曲线、共鸣深度 |
| **市场潜力** | 类型定位、目标受众、竞品对标、商业可行性 |

### 可视化输出

- **雷达图**：五维综合评分一目了然
- **人物关系图谱**：角色关系拓扑网络
- **情绪节奏曲线**：情节张力的可视化波形
- **市场对标看板**：同类作品对比分析
